# AI-Powered-Email-and-Order-Management-System

This project automates customer email processing by classifying emails, handling orders, and generating product recommendations using AI and NLP techniques.

Email Classification:
Uses a pre-trained NLP model to categorize emails into product inquiries or order requests, enabling efficient task automation.

Order Processing:
Extracts product IDs and quantities from order-related emails, checks stock availability, updates inventory, and generates appropriate customer responses.

Product Recommendations:
Implements a TF-IDF-based search to find relevant products for customer inquiries and generates personalized product suggestions.

Response Generation:
Automatically crafts personalized responses based on classified emails and product recommendations, saving them in a CSV format.
